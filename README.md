# Macro-Phyton
## Python – Análisis de datos macroeconómicos (BCRA)

Proyecto en Python que descarga y procesa datos públicos del BCRA y REM,
trabaja con series temporales y genera visualizaciones del tipo de cambio
oficial junto a bandas cambiarias.

### Qué hace el proyecto
- Descarga datos oficiales (BCRA / REM)
- Limpia y combina series con distintos calendarios
- Visualiza resultados con matplotlib

### Tecnologías
- Python
- pandas
- matplotlib
